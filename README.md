TMUX
----

- Quit from command: `q`
- Start selection mode: `Ctrl-b [`
- Copy: `y`
- Paste: `Ctrl-b p`
- Help: `Ctrl-b ?`
- Split horizontal: `Ctrl-b %`
- Split vertical: `Ctrl-b "`
- Escape selection mode: `Esc-Enter`
- Join panes: `join-pane (-h) -s1 -t0`
- break pane: `Ctrl-b !`
